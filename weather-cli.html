Weather CLI

Python · API · CLI

Aplikasi cuaca terminal yang mengambil data dari OpenWeatherMap API secara real-time. Cukup ketik nama kota, langsung dapat info cuaca lengkap di terminal.

GitHub →

Fitur

  • Cuaca real-time dari API
  • Input nama kota bebas
  • Tampil suhu, kelembaban, angin
  • Output warna di terminal

Tech Stack

  • Python 3
  • OpenWeatherMap API
  • requests library
  • argparse (CLI args)

Yang Dipelajari

  • Konsumsi REST API
  • Parsing JSON response
  • Bikin CLI interaktif
  • Error handling API
